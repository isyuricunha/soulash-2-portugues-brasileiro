# Soulash 2 em Português Brasileiro

Tradução completa do **Soulash 2** para o Português Brasileiro, cobrindo todos os textos de interface, itens, habilidades, histórias e diálogos do jogo.

> Participe da comunidade no Discord: [discord.com/invite/aYbWBr4](https://discord.com/invite/aYbWBr4)

---

## Instalação via Steam

1. Acesse a [página do mod no Steam Workshop](https://steamcommunity.com/workshop/filedetails/?id=3704275307).
2. Clique no botão verde **Inscrever-se**.
3. Abra o **Soulash 2**.
4. No menu inicial, clique em **Mods**.
5. Localize **Português Brasileiro** na lista e clique em **Ativar**.
6. Vá em **Options**, selecione **Português Brasileiro** na opção de linguagem e salve.

---

## Instalação Manual (outras versões)

1. Baixe os arquivos deste repositório pelo botão **Code > Download ZIP** ou via `git clone`.
2. Navegue até a pasta de instalação do jogo. Exemplos comuns:
   - `C:\Program Files (x86)\Steam\steamapps\common\Soulash 2`
   - `E:\SteamLibrary\steamapps\common\Soulash 2`
   - `C:\Games\Soulash 2`
3. Dentro da pasta do jogo, acesse `data\mods`.
4. Copie a pasta `ptbr` (baixada neste repositório) para dentro de `data\mods`.
5. Abra o jogo, vá em **Mods** no menu inicial e ative **Português Brasileiro**.
6. Vá em **Options**, selecione **Português Brasileiro** e salve.

---

## Observações

- O jogo gera nomes de itens e lugares aleatoriamente ao criar um novo mundo. Se você iniciar uma partida em Inglês e depois mudar para Português, alguns nomes já gerados podem permanecer no idioma original.
- **Recomenda-se criar um novo mundo com a tradução já ativa** para que tudo seja gerado corretamente em Português Brasileiro. Mas não é obrigatório — a maior parte do jogo ficará traduzida de qualquer forma.
